# New MyTube

## :tv:&nbsp; 소개
- Youtube가 제공하는 APIs를 사용하여 인기 동영상, 검색한 동영상, 연관 동영상을 보여주는 커스터마이즈된 MyTube 구현
- 리엑트 Context API, TanStack Query, React Router 등의 활용 
- Test Code 작성 &nbsp; :point_right: &nbsp; 컴포넌트 테스트, E2E 테스트 실행
